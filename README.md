# Football Simulation

Public source-code repository for an open-source football simulation.

## Status

The project is in its foundation phase. No game, physics engine, renderer, machine-learning model or gameplay implementation has been published yet.

This repository is the authoritative publication surface for source code intended for public use and contribution. Internal planning material is not distributed here and is not required to build, test, study or modify any code that is published here.
